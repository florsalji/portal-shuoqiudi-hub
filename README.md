# portal-shuoqiudi-hub

## 项目简介

`portal-shuoqiudi-hub` 是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定域名或具体网站，仅作为 HTML 页面集合的存储与展示空间。

## 目录结构

```
portal-shuoqiudi-hub/
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 页面引用的资源文件（图片、样式、脚本等）
├── index.html      # 仓库入口页面（可选）
└── README.md       # 本文件
```

各目录说明：

- `pages/`：每个子目录或文件代表一个独立页面，文件名即页面标识。
- `assets/`：按需存放页面所需的静态资源，建议按页面或类型组织子目录。

## 页面归档说明

本仓库收录的 HTML 页面均为独立自包含的网页文件。每个页面可单独运行，不依赖外部服务或特定域名。页面内容可能涉及信息展示、工具、说明文档等，具体用途请查看对应页面内的说明。

页面归档遵循以下原则：

- 每个页面保持独立，不互相依赖。
- 页面内资源尽量内联或存放在 `assets/` 中，避免外部链接。
- 页面命名应具有可读性，便于识别内容。
